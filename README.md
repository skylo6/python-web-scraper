# python-web-scraper
Python web scraping tool that extracts, cleans, and exports structured data (CSV &amp; JSON) from websites using BeautifulSoup and pandas.
This project is a Python-based web scraping and data processing tool designed to extract structured information from websites.

It collects book data such as title, price, and rating from a sample e-commerce site, then processes and cleans the data using pandas. The cleaned dataset is exported into CSV and JSON formats for easy analysis and storage.

The project demonstrates key data handling skills including web scraping, data cleaning, transformation, and structured data output, simulating a real-world data pipeline workflow.
